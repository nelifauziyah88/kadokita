# KadoKita

KadoKita adalah website untuk membuat kado ucapan online yang bisa dibagikan kepada teman, saudara, rekan kerja, atau pasangan. Dengan KadoKita, kamu bisa membuat ucapan spesial secara digital dan membagikannya dengan mudah melalui link.

## Demo Website
Coba langsung websitenya di sini:  
👉 https://kadokita.my.id/

## Fitur
- Membuat ucapan kado secara online
- Tampilan sederhana dan mudah digunakan
- Bisa dibagikan melalui link
- Cocok untuk berbagai momen spesial

## Teknologi
Project ini dibuat menggunakan teknologi web seperti:
- HTML
- CSS
- JavaScript

## Cara Menjalankan Project
1. Clone repository ini
```bash
git clone https://github.com/username/nama-repo.git
```
2. Masuk ke folder project
```bash
cd nama-repo
```
3. Buka file index.php di browser
